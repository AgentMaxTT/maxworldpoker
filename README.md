# TT Poker 官方网站

这是 TT Poker 扑克游戏的官方网站，托管在 GitHub Pages 上。

## 网站功能

- 游戏介绍和特点展示
- iOS 和 Android 应用下载
- 安装说明和常见问题解答
- 联系方式和重要声明

## 技术栈

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (原生)
- GitHub Pages 托管

## 部署说明

1. 将本仓库设置为 GitHub Pages 源
2. 配置自定义域名：maxworldpoker.com
3. 更新咕噜分发平台的下载链接
4. 定期更新维护信息

## 文件结构

```
├── index.html          # 主页面
├── CNAME              # 自定义域名配置
├── README.md          # 说明文档
└── assets/            # 静态资源（可选）
```

## 维护说明

1. 更新下载链接：修改 index.html 中的链接地址
2. 更新维护信息：修改页面顶部的维护提示
3. 更新联系方式：修改邮件地址和联系方式
4. 定期备份：GitHub 会自动版本控制

## 注意事项

- 网站内容需符合相关法律法规
- 明确标注游戏的娱乐性质
- 禁止任何赌博相关描述
- 添加年龄限制提示（18+）
